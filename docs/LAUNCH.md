## Short announcement

> ConvexKit 1.0 is out: create a production-minded TanStack Start + Convex app with one command.
> Choose Better Auth or Clerk, Cloudflare/Vercel/Netlify, Terraform, and only the examples you need.
> The generated app includes realtime data, strict tests, browser smoke coverage, and working AI,
> Stripe, email, upload, RBAC, forms, chat, and Todos examples.

```bash
npm create convexkit@latest my-app
```

## Submission checklist

- Convex templates gallery: include the demo, repository, one-line pitch, and banner.
- TanStack community examples: emphasize TanStack Start SSR, Router, Query, and Table usage.
- Convex and TanStack Discord: use the short announcement and one screenshot/GIF.
- r/reactjs: include architecture tradeoffs and disclose that this is the maintainer's project.
- dev.to/blog: expand the architecture sequence, zero-OAuth first run, and provider composition.
- Awesome lists: submit narrowly to lists that already accept TanStack, Convex, or Cloudflare tools.

## Metrics

Review monthly after launch:

- GitHub stars, forks, unique visitors, clones, and template uses
- npm weekly downloads for `create-convexkit`
- Docs traffic and top entry pages
- Demo sign-ups and successful authenticated E2E runs
- Issues labeled onboarding, deployment, or generator
